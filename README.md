                                        ТЕХНИЧЕСКОЕ ЗАДАНИЕ
                                  на разработку приложения "Сапёр"
  Целью явлется создание известной игры сапёр, в которой пользователь должен открыть все безопасные клетки, избегаю клеток с минами. Структура строится из нескольких классов. Класс Cell содержит следующие поля: bool Mine (наличие мины), int cntNeighbors (количество мин вокруг клетки), bool UseCell (открытая клетка), bool CellFlagged (клетка отмечена флагом). Следующий класс должен содержать в себе методы, отвечающие за логику игры: инициализация поля, размещение мин рандомным образом, проверка на победу, подсчет количества мин вокругт каждой клетки. Класс содержит в себе поля: int size (размеро игрового поля), int mines (количество мин), cells[,] field (игровое поле). Алгорти работы. При запуске программы должно создаться игровое поле, на котором должны быть размещены рандомным образом заданное количество мин. Пользователь поочерёдно открывает клетки. При нажатии на клетку с миной пользователь проигрывает, если нет то ему отображатся число мин вокруг нажатой клетки. Программа содержит две формы. Основная форма (MainForm) содержит: игровое поле (реализованное через массив кнопок button), кнопку новая игра, отображение текущего состояние игры. Вторая форма (ResultForm) предназначена для отображения итогов игры. Содержит: сообщение о резултьтате игры, количество ходов, кнопки "играть снова" и "выход".
