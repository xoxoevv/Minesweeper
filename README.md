                                        ТЕХНИЧЕСКОЕ ЗАДАНИЕ
Разрабатывается Windows Forms приложение на языке программирования C# для учета и контроля выполнения привычек пользователя. В проекте используются два класса: Habbit, HabbitManager. Класс Habit будет содержать следующие поля: string name (название привычки), bool IsDone (статус выполнения привычки), в последствии возможно и другие. Класс HabitManager должен содержать: List<Habit> Habits (список привычек), а также несколько методов, которые будут использоваться для добавления привычки, удаления привычки, изменения статуса выполнения и сохранения списка в файл (пока что неизвестного формата). Приоложение должно содержать две формы: MainForm, AddHabitForm. Первая содержит следующие элементы: ListBox или DataGridView для отображения списка привычек. Кнопки управления:
 • Добавить привычку
 • Удалить привычку
 • Отметить выполнение
 • Сохранить данные
 • Загрузить данные
Функции формы:
 • отображение списка Habit
 • вызов методов HabitManager
Вторая форма содержит: TextBox для ввода названия привычки, кнопка подтверждения добавления и возможно (DateTimePicker для выбора даты). Функцию оздание объекта Habit и передача его в MainForm. Логика состоит в том, что пользователь добавляет привычку через AddHabitFormб далее создается объект класса Habit. Объект добавляется в список HabitManager. Данные отображаются в MainForm.
